Introduction
The Expense Tracker is a simple web application that allows users to track their income and expenses. It provides features such as adding and removing transactions, filtering by category, exporting data, importing data, and visualizing expenses using a pie chart.

Features
-Add Transactions: Record your income and expenses with details like date, description, category, and amount.
-Filter by Category: View transactions by specific categories like Food, Transportation, etc.
-View Summary: See a summary of your total income, total expenses, and net income.
-Visualize Expenses: View a pie chart of your expenses broken down by category.
-Export Data: Save your transactions as a JSON file.
-Import Data: Load transactions from a JSON file.

How to Use
	1)Clone the Repository: Clone the repository to your local machine.
	2)Open index.html: Open the index.html file in your web browser to start using the Expense Tracker.
	3)Add a Transaction:
		Enter the date, description, category, and amount.
		Click "Add Transaction" to save the entry.
	4)Filter Transactions:
		Use the dropdown to filter transactions by category.
	5)View Summary and Chart:
		The summary section updates with your total income, expenses, and net income.
		The chart visualizes your expenses by category.
	6)Export Transactions:
		Click "Export Data" to save your transactions as a JSON file.
	7)Import Transactions:
		Click "Import Data" and select a JSON file to load transactions.
Requirements
-A modern web browser (Chrome, Firefox, Edge, etc.)
